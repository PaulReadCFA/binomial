Binomial Option Pricing Equations

The value of a call option at t = 1 once it can be exercised is:

$C_u = \max(\color{#047857}{S_u} - K, 0)$ in the up state

$C_d = \max(\color{#b91c1c}{S_d} - K, 0)$ in the down state

The value of a put option at t = 1 once it can be exercised is:

$P_u = \max(K - \color{#047857}{S_u}, 0)$ in the up state

$P_d = \max(K - \color{#b91c1c}{S_d}, 0)$ in the down state

From these option values the hedge ratio for the call (HRC) and the hedge ratio for the put (HRP) can be computed by combining the stock with the appropriate option so that the combination becomes riskless:

$$HR_C = \frac{C_u - C_d}{\color{#047857}{S_u} - \color{#b91c1c}{S_d}}$$

$$HR_P = \frac{P_u - P_d}{\color{#047857}{S_u} - \color{#b91c1c}{S_d}}$$

This now allows us to compute the call and the put value at t = 0 as follows:

$$c_0 = \color{#047857}{S_0} \times HR_C - \frac{HR_C \times \color{#047857}{S_u} - C_u}{1 + r}$$

$$p_0 = \color{#047857}{S_0} \times HR_P - \frac{HR_P \times \color{#047857}{S_u} - P_u}{1 + r}$$
